# FakeBook

A simple social media clone built with HTML, CSS, and JavaScript.

## Features
- Responsive navbar
- User info modal (shows name, username, email, etc.)
- Create posts with text and images
- Image preview and filename display
- New posts appear at the top

## Setup
1. Clone or download the project.
2. Open `index.html` in your browser.
3. Make sure the `assets` folder is in the right place.

## Usage
- Click the profile icon to see user info.
- Type in the textarea or upload an image to enable the Post button.
- Click Post to publish.  
- Posts show your name, time, text, and image.
